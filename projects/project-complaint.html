SI-Lapor – Sistem Informasi Laporan Masyarakat

2024 · Laravel 12 · Kelurahan e-Government

SI-Lapor adalah aplikasi web pelaporan masyarakat untuk lingkungan kelurahan yang memudahkan warga membuat pengaduan, memantau status, dan berkomunikasi dengan admin melalui antarmuka modern dan terstruktur.

Laravel 12 Laravel Breeze Tailwind CSS MySQL Role-based Access

Project overview

SI-Lapor memusatkan proses pengelolaan pengaduan di kelurahan: mulai dari pembuatan pengaduan oleh warga, verifikasi dan penanganan oleh admin, hingga dokumentasi komentar dan lampiran secara rapi.

  • Sistem pelaporan terintegrasi: warga bisa membuat dan memantau pengaduan sendiri.
  • Manajemen status pengaduan (baru, diproses, selesai) dengan riwayat tanggapan.
  • Dukungan komentar dan lampiran file pada setiap pengaduan.
  • Multi-user & multi-level akses (warga dan admin kelurahan).

Role

Perancangan flow pengaduan end-to-end, autentikasi, role-based access, dan logika backend utama.

Tech

  • Laravel 12, Laravel Breeze (Blade).
  • Tailwind CSS, komponen layout responsif.
  • MySQL, relasi terstruktur untuk users, pengaduan, komentar, dan tanggapan.
