Camping Rental System — Highlight

Aplikasi rental alat camping untuk Bontang Outdoor dengan alur booking yang jelas dari katalog, form pemesanan, hingga bukti PDF dan panel admin untuk mengelola stok serta status pembayaran.

Fitur utama

  • Katalog alat dengan harga per hari dan stok.
  • Form booking multi-langkah tanpa perlu login.
  • Perhitungan biaya otomatis berdasarkan durasi sewa.
  • Halaman sukses + bukti booking dalam bentuk PDF.

Panel admin

  • Dashboard ringkas jumlah booking & status.
  • CRUD alat (foto, harga, stok, status aktif).
  • Manajemen status booking (pending, approved, returned).

Project overview

User dapat melihat list alat outdoor yang tersedia, melakukan booking, dan memilih tanggal ambil serta tanggal pengembalian. Sistem menghitung biaya otomatis berdasarkan durasi sewa.

  • List alat outdoor dengan stok dan kondisi.
  • Form booking dengan validasi tanggal.
  • Perhitungan biaya dan ringkasan sewa.

Role

Fullstack Laravel (UI sederhana + backend logic).

Tech

  • Laravel, Livewire.
  • MySQL.
  • Basic validation & business logic.
