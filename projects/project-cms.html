CMS Portal Berita (Filament)

2023 · Laravel & Filament

CMS portal berita yang menggunakan Filament Admin Panel untuk mengelola artikel, kategori, dan user (penulis/editor/admin).

Laravel Filament MySQL

Project overview

Dengan Filament, admin bisa mengelola konten berita lewat UI yang modern tanpa perlu membuat dashboard dari nol. Portal publik menampilkan daftar berita dan halaman detail artikel.

  • CRUD artikel, kategori, dan user di Filament.
  • Role-based access untuk penulis, editor, dan admin.
  • Halaman publik untuk list dan detail berita.

Role

Setup Filament, definisi resource, dan integrasi front-facing portal.

Tech

  • Laravel, Filament.
  • MySQL.
  • Authentication & authorization.
