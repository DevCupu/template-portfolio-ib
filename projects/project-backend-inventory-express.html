Inventory REST API

Backend Inventory REST API (Express)

Latihan backend · Manajemen stok barang

API inventory ini saya bangun untuk mengelola data stok barang: mulai dari menambah item baru, memperbarui jumlah stok, hingga menandai barang yang hampir habis. Project ini membantu melatih logika bisnis sederhana di sisi backend.

Express REST API Node.js

Fitur utama

  • CRUD data item barang.
  • Update stok masuk dan keluar.
  • Penandaan barang low-stock berdasarkan threshold.

Tech stack

  • Node.js + Express.
  • Database (contoh: MongoDB/MySQL) sesuai kebutuhan latihan.
  • Postman untuk pengujian endpoint.
